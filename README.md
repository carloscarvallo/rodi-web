rodi-web
========

Will be a web-service interface for RoDi (Robot Didáctico inalámbrico).

Testing
-------

Via browser go to:

```
 http://192.168.4.1:1234/whatever
```

Or, via curl:

```
curl --get http://192.168.4.1:1234/whatever --verbose
```
